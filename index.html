onenote-replace-calibri-with-mono

发表于 2020-05-26 分类于 Windows
本文字数： 1.2k 阅读时长 ≈ 1 分钟

1. 解决方案

发现一个字体格式 Microsoft YaHei Mono，在中文时使用微软雅黑，在英文时使用 Consolas。因此将默认字体切换为 Mono，输入中文后再输入的英文会是 Consolas。

但是直接输入英文，OneNote 会切换为 Calibri，所以需要将 Calibri 字体替换为 Mono。

2. 工具准备

  • Mono 字体/archives/4100fdb5/MSYHMONO.ttf
  • FontCreator 软件，用于修改字体名称，下载链接

3. 字体替换

  • 备份 Calibri 字体，到 C:\Windows\Fonts 目录直接拖动 Calibri 到备份文件夹
  • 安装 Mono 字体，直接右键安装
  • 替换 Calibri 字体为 Mono
    • 运行 FontCreator.exe，输入 FontCreator 的注册码
    • 将 Mono 字体拖入到 FontCreator 界面中
    • 点击 FontProperties，修改 Font Family 为 Calibri
    • 点击 File 》Export Font 》Export TrueType/OpenType Font 导出修改后的字体
    • 右键安装导出的字体

OneNote 操作

点击 文件 》选项 》常规，修改默认字体为 Microsoft YaHei Mono，之后中文字体会使用微软雅黑，英文字体使用 Consolas。
